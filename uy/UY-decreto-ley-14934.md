# Decreto Ley Nº 14934 - APROBACION DEL CONVENIO MULTINACIONAL PARA LA PROSECUCION DE LAS ACTIVIDADES DEL CENTRO REGIONAL DE SISMOLOGIA PARA AMERICA DEL SUR

##### Artículo 1

Apruébase el Convenio Multinacional para la Prosecución de las
Actividades del Centro Regional de Sismología para América del Sur
(CERESIS), hecho en Lima (Perú), el 18 de junio de 1971. (*)

Comuníquese, etc.
